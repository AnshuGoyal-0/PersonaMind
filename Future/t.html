Section 2: Short Answer Questions (5 points each)

  1. Briefly explain the difference between hardware and software.
    Hardware refers to the physical components of a computer system, such as the monitor, keyboard, mouse, CPU, RAM, and hard drive. You can see and touch hardware. Software is a collection of instructions that tell the hardware what to do. Software is intangible and cannot be physically touched.
  2. Describe the function of the Central Processing Unit (CPU) in a computer.
    The Central Processing Unit (CPU) is often referred to as the "brain" of the computer. It is responsible for processing instructions and data. It fetches, decodes, and executes instructions from software programs.
  3. What are the different types of input devices used with computers? Give two examples.
    Input devices are used to enter data and instructions into a computer. Here are two examples:
    • Keyboard: Used for typing text and entering commands.
    • Mouse: Used to point, click, and drag objects on the screen.
    There are many other input devices, such as scanners, webcams, microphones, joysticks, etc.
  4. Explain the importance of using strong passwords.
    Strong passwords are important for computer security. They help prevent unauthorized access to your computer and data. A strong password should be:
    • At least 8 characters long
    • A combination of uppercase and lowercase letters, numbers, and symbols
    • Not a word found in a dictionary or easily guessable personal information (like your birthday)
  5. Briefly explain what the term "cloud computing" refers to.
    Cloud computing refers to the delivery of computing services (like storage, servers, databases, software, etc.) over the internet. Users can access these services on-demand from anywhere with an internet connection. They don't need to manage the physical hardware or software themselves.

Bonus Question (3 points):

List three basic computer maintenance tasks you can perform to keep your computer running smoothly.

  • Regularly update your operating system and software. Updates often contain security patches and bug fixes.
  • Restart your computer periodically. This can help clear temporary files and improve performance.
  • Run a disk cleanup utility. This can help free up storage space by removing unnecessary files.

1. Basic Questions:

  1. What are the main components of a computer (e.g., CPU, monitor, keyboard)?
    Answer: The main components of a computer include:
    • CPU (Central Processing Unit): The brain of the computer that performs instructions defined by software.
    • Monitor: The screen that displays visual output from the computer.
    • Keyboard: An input device used to type text and commands into the computer.
    • Mouse: An input device used to interact with the graphical user interface.
    • RAM (Random Access Memory): Temporary storage that the CPU uses for processing tasks.
    • Hard Disk (or SSD): Permanent storage for the operating system, software, and files.
    Example: The CPU processes data, the monitor displays it, and the keyboard and mouse allow the user to interact with the computer.
  2. What is an operating system? Name a few examples.
    Answer: An operating system (OS) is software that manages the hardware and software resources of a computer and provides common services for computer programs. Examples include:
    • Windows: Developed by Microsoft.
    • macOS: Developed by Apple.
    • Linux: An open-source OS.
    • Android: Developed by Google for mobile devices.
    • iOS: Developed by Apple for iPhones and iPads.
    Example: Windows 10, macOS Catalina, and Ubuntu Linux are all operating systems.
  3. How do you properly shut down a computer?
    Answer: To properly shut down a computer:
    • Windows: Click on the Start button, select Power, and then choose Shut down.
    • macOS: Click on the Apple menu and select Shut Down.
    • Linux: Depending on the distribution, you may find a shutdown option in the system menu or use a terminal command like sudo shutdown now.
    Example: In Windows 10, click the Start button, click the Power icon, and select Shut down.
  4. What is the function of a web browser?
    Answer: A web browser is a software application used to access and view websites on the internet. It interprets HTML, CSS, and JavaScript code from web servers to display text, images, and other content on your screen. Examples of web browsers include Google Chrome, Mozilla Firefox, Microsoft Edge, and Safari. Example: Using Google Chrome to visit and interact with websites like www.google.com.

2. Intermediate Questions:

  1. Explain the difference between hardware and software.
    Answer:
    • Hardware: The physical components of a computer that you can touch and see, such as the CPU, monitor, keyboard, mouse, hard drive, and RAM.
    • Software: The intangible instructions and programs that run on the hardware, enabling it to perform specific tasks. This includes operating systems, applications, and drivers.
    Example: The monitor (hardware) displays the content generated by Microsoft Word (software).
  2. How do you install a new software application?
    Answer: To install a new software application:
    • Download the installer from the software’s official website or insert the installation media (e.g., a CD or USB drive).
    • Run the installer file, usually named something like setup.exe or install.exe.
    • Follow the on-screen instructions, accepting the license agreement and choosing an installation location if necessary.
    • Once installation is complete, open the application to verify it’s working correctly.
    Example: Downloading and installing Google Chrome by visiting www.google.com/chrome, clicking "Download," and following the instructions.
  3. What is a file extension? Give examples of common file extensions and their associated file types.
    Answer: A file extension is the suffix at the end of a filename that indicates the file type. It is typically composed of three or four letters and comes after a period (e.g., .txt, .docx, .jpg). Examples:
    • .txt - Plain text file (opened with Notepad or TextEdit).
    • .docx - Microsoft Word document.
    • .jpg - JPEG image file (viewable in most photo viewers).
    • .mp3 - Audio file (played by music software).
    Example: A photo may be saved as picture.jpg, indicating it is an image in JPEG format.
